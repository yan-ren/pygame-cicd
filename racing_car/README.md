# Racing Car
A simple racing car game written in Python and PyGame

## Dependency Requirements
- Python3.7
- PyGame1.9.6
- Windows10 / Ubuntu 16.04.6 LTS

## Implementation Notes
- mp3 format is not working on Ubuntu, use wav instead
